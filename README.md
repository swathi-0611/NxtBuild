# NxtBuild 🚀
### Live Demo: [https://nxt-build-two.vercel.app](https://nxt-build-two.vercel.app)



NxtBuild is a full-stack platform that transforms natural language prompts into functional, production-ready web components. By leveraging Large Language Models, it automates the "idea-to-code" workflow for developers and designers.

---

## ✨ Key Features
* **AI Code Generation:** Integrated with **Google Gemini API** to generate structured HTML, CSS, and JavaScript.
* **Real-time Sandbox:** Instantly renders generated code in a secure preview window for immediate feedback.
* **Responsive Previews:** View how the generated code looks on different screen sizes.
* **Modern UI:** Built with a sleek, dark-themed interface using **Tailwind CSS**.

---

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite
* **Styling:** Tailwind CSS, Lucide React (Icons)
* **AI Engine:** Google Gemini SDK
* **Deployment:** Vercel (Frontend) & CI/CD via GitHub Actions
